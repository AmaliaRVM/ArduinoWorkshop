Arduino & Elements :
Arduino is an open-source electronics platform based on easy-to-use hardware and software. It makes it as easy as possible to program tiny computers called microcontrollers. Arduino can make your projects responsive, was design to help you get things done. To make that happen, they kept the background material on programming and electronics to a minimum..
×
Circuit :
For anything to happen electrically, you have to have a complete circuit. This is a fundamental principle. A good example is a flashlight. The first drawing shows a complete circuit. A complete circuit consists of a power source (battery), a load (lamp) a connection from the power source to the load and a connection from the load back to the power source. Electricity has a path from the voltage to the load and from the load back to the source making a complete circuit. The Arduino's circuit are build on the Breadboard.
×
Code :
The Arduino Integrated Development Environment (IDE) is the main text editing program used for Arduino programming. It is where you'll be typing up your code before uploading it to the board you want to program. Arduino code is referred to as sketches (nale of the files) and is based on the C++ language. (Note: It's important to use the latest version of Arduino IDE). You can write your own code or use the library already written by the wide Arduino's community.
